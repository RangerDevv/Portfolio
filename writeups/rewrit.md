## The Problem

StackOverflow is great, but it can feel intimidating for beginners. The UX hasn't kept up with modern developer expectations — no real-time collaboration, no native Markdown editing, and a rigid Q&A format that doesn't lend itself to tutorials or long-form documentation.

## The Solution

ReWrit reimagines developer knowledge-sharing with a Markdown-first editor, threaded commenting, community ratings, and a clean Svelte-powered UI. It's designed to feel fast, accessible, and welcoming to developers of all skill levels.

## Tech Stack

- Svelte / SvelteKit
- TailwindCSS
- Markdown (MDsveX)
- Vercel Deployment

## Results

- **5** Stars
- **1** Forks
- **Active** Live Users

## Lessons Learned

Building a content platform taught me a lot about editor UX, real-time state management, and how important good typography and spacing are for readability. Svelte's reactivity model made the real-time preview a breeze to implement.
