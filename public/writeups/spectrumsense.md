## The Problem

Early detection of autism spectrum disorder (ASD) is critical for effective intervention, but current screening methods can be slow, subjective, and inaccessible to many families.

## The Solution

SpectrumSense uses machine learning models trained on behavioral datasets to provide early indicators of ASD. The tool is designed to be a supportive screening aid — not a replacement for professional diagnosis.

## Tech Stack

- Python
- Scikit-learn
- Pandas
- PyTorch

## Lessons Learned

Working on a healthcare-adjacent project reinforced the importance of ethical AI — being transparent about model limitations, avoiding overconfident predictions, and centering the user's wellbeing above technical novelty.
